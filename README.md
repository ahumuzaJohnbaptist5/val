# Val Executive Restaurant

Single-page HTML website for Val Executive Restaurant with an embedded Dialogflow ES chatbot widget.

## File

- `restaurant_website.html` — complete website (HTML + embedded CSS).

## Local preview

Open the file directly in a browser:

```bash
open restaurant_website.html
```

(Or double-click the file on Windows/Linux/macOS.)

## Deploy

1. Upload `restaurant_website.html` to your host (GitHub Pages, Netlify, cPanel, etc.).
2. Configure your custom domain (for example `valexecutiverestaurant.com`).
3. Replace `YOUR_DIALOGFLOW_AGENT_ID` in the HTML file with your Dialogflow Messenger agent ID.
